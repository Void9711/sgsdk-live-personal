# SGSDK Live

Live repository for SGSDK.

## jenkins

Pipeline as Code, using [Jenkins Pipeline](https://www.jenkins.io/doc/pipeline/).

## terraform

Infrastructure as Code, using [Terraform](https://terraform.io/) and [Terragrunt](https://github.com/gruntwork-io/terragrunt).

## nomad

Deployment as Code, using [Nomad](https://www.nomadproject.io/).
